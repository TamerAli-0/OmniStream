# Milestones: OmniStream

## v1.0 — Core App (Completed)

**Shipped:** 2026-01-29
**Phases:** 1-7 (pre-GSD, tracked in DEVLOG.md and PROJECT_SPEC.md)

### What shipped:
- Multi-source content aggregation (manga, anime, movies)
- MangaDex, AnimeKai (WebView), FlickyStream, WatchFlix, AsuraComics, MangaKakalot, ManhuaPlus, VidSrc, Goojara
- Manga reader with vertical scroll + chapter navigation
- Video player with HLS/DASH + AES decryption (Vidzee, Cloudnestra)
- Two-tier access gate (VIP + Standard)
- Auth system (JWT) with Node.js backend on Render + MongoDB Atlas
- Room database for favorites
- Settings with 8 color themes + dark mode
- Library with cloud sync
- Unified search
- Home + Browse screens
- TMDB API key security

### Known issues carried forward:
- Library doesn't show local favorites
- No reading progress persistence
- GogoAnime details crash
- Search race conditions
- Downloads placeholder
