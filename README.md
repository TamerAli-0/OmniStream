# OmniStream

A unified Android streaming app for manga and video content. Built with modern Android development practices using Kotlin and Jetpack Compose.

## Download

**[Download Latest APK](https://github.com/TamerAli-0/OmniStream/releases/latest)**

> Requires Android 7.0 (API 24) or higher

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Language | Kotlin |
| UI | Jetpack Compose + Material 3 |
| Architecture | MVVM + Clean Architecture |
| DI | Hilt (Dagger) |
| Database | Room |
| Networking | OkHttp + Jsoup |
| Video | Media3 ExoPlayer (HLS, DASH) |
| Image Loading | Coil |
| Async | Kotlin Coroutines + Flow |
| Storage | DataStore Preferences |
| Background | WorkManager |

## Features

- Manga reader with chapter tracking
- Video player with HLS and DASH support
- Favorites and watch history (Room database)
- Background downloads via WorkManager
- Material 3 dynamic theming
- Smooth Compose animations and transitions

## Building from Source

1. Clone the repo
   ```
   git clone https://github.com/TamerAli-0/OmniStream.git
   ```
2. Open in Android Studio (Hedgehog or newer)
3. Sync Gradle and run on device/emulator

## Installation

1. Download the APK from [Releases](https://github.com/TamerAli-0/OmniStream/releases/latest)
2. Enable "Install from unknown sources" on your Android device
3. Open the APK and install
